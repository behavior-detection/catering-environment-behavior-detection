# catering-environment-behavior-detection
## 使用说明
前端：

        1.'npm install'
        
        2.`npm run dev`

后端：

        1. 创建虚拟环境
                window： python -m venv venv
                macOS / Linux：python3 -m venv venv
        
        2. 激活虚拟环境
                Windows： venv\Scripts\activate
                macOS / Linux：source venv/bin/activate
        
        3. 安装依赖
                pip install -r requirements.txt
